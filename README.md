# CPU Trace Visualization Tool

## 📌 Overview
This project visualizes CPU execution traces using an interactive Gantt chart built with Python, Pandas, and Plotly.

---

## 🚀 Features
- CPU execution timeline visualization
- Regex-based stage grouping
- CPU active and idle time analysis
- Interactive Plotly Gantt chart
- Exportable HTML dashboard

---

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly
- Jupyter Notebook

---

## 📁 Dataset
Dataset contains system trace logs. A sample structure is provided for demonstration. Actual data is not included due to confidentiality.
